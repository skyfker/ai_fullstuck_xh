# 项目开发步骤
需求分析->设计文档->开发->测试->部署

我的开发过程如下

- 静态资源的引入
- 创建路由--通过插件 vue-router 来管理 *
    导入函数
    配置路由规则
    创建路由对象
    挂载路由对象
- 创建组件
    使用element plus 组件库。引入方法：
    完整导入
    按需导入
    自动导入（推荐）
- 配合路由增加页面


- axios 的封装
